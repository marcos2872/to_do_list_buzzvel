## Getting Started

### Prerequisites

- [NodeJs](https://nodejs.org/en)

### Installing

install the dependencies

    npm install


Running in developer mode, will run on port 3000

    npm run dev

Running in production mode, will run on port 3000

    npm run build
    npm start


## Running the tests

### Sample Tests

run the command below

    npm test
